e115: Supported Filetypes

e115 supports the following file types upon upload.

Type Deprecated Maximum Filesize
.png No 100Mib
.jpeg No 100Mib
.gif No 20Mib
.apng No 20Mib
.webm No 100Mib
.mp4 No 100Mib
.swf (Flash) Yes
.sb3 (Scratch) No 100Mib
